## 关于金砖华荣
深圳金砖华荣资产管理有限公司（中基协私募基金管理人编号：P1014425）成立于2015年1月，金砖华荣投资研究团队来自于银行、信托、证券、 国际投行公司，公司凭借投研团队的专业能力和资源优势，对金融、房地产、智能科技等成长性行业 和板块进行深入的研究。未来公司将精准把握大陆多层次资本市场、房地产市场，持续创造佳绩。
